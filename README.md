# CommitNote

## 구현 할 기능 목록

### 노트 저장 기능

- [ ] 사용자가 제목과 내용을 입력한 후 저장 버튼을 누르면 해당 내용을 저장한다
    - [x] 저장 버튼을 만든다
- [x] 사용자로부터 텍스트를 입력받는다
- [x] 문자열에 SHA-1을 통해 해시값을 얻는다
- [ ] 노트 제목과 내용에 구분자를 포함해 SHA-1 해싱을 적용한다
- [ ] 해싱이 적용된 파일을 저장한다(blob의 역할을 하는 파일을 저장한다)
- [ ] blob 역할 파일의 해시값과 수정 시간, 부모 커밋 객체를 가지는 데이터를 만든다
    - [ ] 최초의 저장이라면 부모 커밋 객체는 추가하지 않는다
    - [ ] 만들어진 데이터에 SHA-1 해싱을 적용한다
    - [ ] 적용된 파일을 저장한다(commit 객체의 역할을 하는 파일을 저장한다)
- [ ] 가장 최신 노트를 가르키는 HEAD에 만들어진 commit 객체로 설정한다
